# playground

收录练习、小型个人项目和所有不属于特定项目的代码

# 目录说明

```text
playground/
│
├── vue-frontend/       # 前端 vite + vue 项目
│
├── demo-multi/         # Gradle，多项目构建demo
│   └── app             # Gradle，多项目构建demo下的app模块，SpringBoot项目启动点  
│   └── shared          # Gradle，多项目构建demo下的shared模块，包含工具类
├── settings.gradle     # Gradle，定义包含的模块
├── build.gradle        # Gradle，根项目的构建脚本（全局配置）
├── gradle.properties   # 全局属性（版本号等）
```
