postgis，地理空间数据库扩展，存储、查询、分析空间数据（点、线、多边形）的能力，广泛用于地图服务、物流规划、地理位置分析

pgRouting，基于postgis构建，提供图论和路径规划功能（如最短路径、旅行商问题），常用于导航系统

pgvector，存储向量嵌入，执行高效的相似性搜索，是构建RAG（检索增强生成）应用和本地AI知识库核心组件

timescaledb，对时间序列数据优化，自动处理分区、压缩和连续聚合

pg_partman，自动化管理表分区，基于时间或序列id自动创建、维护和清理分区表

uuid-ossp，生成uuid的标准工具，pg13+内置了gen_random_uuid函数，但该扩展仍被广泛使用以兼容旧代码
