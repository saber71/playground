声明式分区
PARTITION BY RANGE，范围分区，如日期、数字区间
PARTITION BY LIST，列表分区，如状态值、地区代码
PARTITION BY HASH，哈希分区，均匀分布数据
优点：语法简洁，数据自动路由，支持主键和唯一索引，查询优化器原生支持
```sql
-- 创建主表（分区表）
CREATE TABLE measurement (
    city_id         int not null,
    logdate         date not null,
    peaktemp        int,
    unitsales       int
) PARTITION BY RANGE (logdate);

-- 创建分区
CREATE TABLE measurement_y2025m01 PARTITION OF measurement
    FOR VALUES FROM ('2025-01-01') TO ('2025-02-01');

CREATE TABLE measurement_y2025m02 PARTITION OF measurement
    FOR VALUES FROM ('2025-02-01') TO ('2025-03-01');
```

继承分区
- 另外需要手动创建触发器或规则将插入数据路由到子表
- 查询使用only或union all才能跨分区
缺点：维护复杂，易出错，性能不如声明式分区
```sql
-- 主表（仅结构，不存数据）
CREATE TABLE measurement (
    city_id         int not null,
    logdate         date not null,
    peaktemp        int,
    unitsales       int
);

-- 子表（分区）
CREATE TABLE measurement_y2025m01 (
    CHECK (logdate >= DATE '2025-01-01' AND logdate < DATE '2025-02-01')
) INHERITS (measurement);

CREATE TABLE measurement_y2025m02 (
    CHECK (logdate >= DATE '2025-02-01' AND logdate < DATE '2025-03-01')
) INHERITS (measurement);
```
