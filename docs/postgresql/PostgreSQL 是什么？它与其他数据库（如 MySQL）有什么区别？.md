对象-关系型数据库管理系统，有着高度的可靠性、可扩展性和严格遵循sql标准

开源免费
高度兼容sql标准
支持高级数据类型，如JSON/JSONB、数组、range范围类型、几何类型、网络地址类型等
强大的扩展能力，支持用户自定义函数（sql、python、c等），可通过扩展（postgis用于地理空间数据、pg_trgm用于模糊搜索）增强功能
完全支持事务与ACID
使用多版本并发控制，读不会阻塞写，写不会阻塞读，极大提升并发性能
支持流式复制、逻辑复制、同步/异步复制

与Mysql的主要区别
mysql的sql标准兼容性较弱，部分语法非标准
数据类型相对基础
postgres的引擎单一但功能强大，mysql有多种引擎需要手动选择
主从复制为主，逻辑复制在较新版本才完善
简单读操作性能快，适合web应用
